Parágrafos e quebras de linhas

Você pode escrever um parágrafo de qualquer jeito, basta colocar tudo no meio do par de tags <p> e </p>

Para fazer uma quebra de linha precisa-se usar a tag <br>

Para usar os sinais <é> preciso usar o código &.l.t; antes e &.g.t; depois, mas sem o ponto final.
